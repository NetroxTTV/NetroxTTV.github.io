📖 Points de grammaire clés

💡 Astuce : Pour l'examen, retiens bien les 3 grands points : le présent poli, le passé, et le formel !

1. Présent poli — 아요 / 어요

Verbe + 아요/어요

가다 → 가요 (je vais)

먹다 → 먹어요 (je mange)

마시다 → 마셔요 (je bois)

Utilisé dans les situations informelles avec des gens qu'on connaît.

2. Passé — 았어요 / 었어요

Verbe + 았어요 (si voyelle ㅏ/ㅗ) ou + 었어요 (autres)

가다 → 갔어요 (je suis allé(e))

먹다 → 먹었어요 (j'ai mangé)

좋다 → 좋았어요 (c'était bien)

재미있다 → 재미있었어요 (c'était amusant)

Pour les noms : N이었어요 / N였어요 = "c'était N"

3. Formel — ㅂ니다 / 습니다 (présent)

Verbe (voyelle finale) + ㅂ니다 / Verbe (consonne finale) + 습니다

가다 → 갑니다

먹다 → 먹습니다

비싸다 → 비쌉니다

Utilisé dans les situations formelles. Passé formel : + 았/었습니다

4. Particule de lieu — 에서

Lieu + 에서 + action

카페에서 커피를 마셔요. → Je bois du café au café.

공원에서 책을 읽어요. → Je lis un livre au parc.

에서 = "à (endroit où se passe l'action)" ≠ 에 = "à (direction / destination)"

5. Connecteur d'actions — 고

Verbe 1 + 고 + Verbe 2 = "et puis"

책을 읽고 커피를 마셔요. → Je lis un livre et bois un café.

손을 씻고 저녁을 먹어요. → Je me lave les mains et mange le dîner.

6. N이에요 / 예요 vs N입니다

N (consonne) + 이에요 / N (voyelle) + 예요

학생이에요. → Je suis étudiant(e). (informel)

요리사예요. → Je suis cuisinier(ère). (informel)

학생입니다. → Je suis étudiant(e). (formel)

✏️ Exercices écrits

💡 Écris ta réponse, puis clique sur "Voir la réponse" pour vérifier !

1. Traduis en coréen : "J'aime les cafés. Donc je vais souvent au café."

저는 카페를 좋아해요. 그래서 카페에 자주 가요.

2. Traduis en coréen : "Hier, je suis allé(e) chez mon ami(e). Nous avons mangé du gâteau et joué à des jeux."

어제 친구 집에 갔어요. 같이 케이크를 먹고 게임을 했어요.

3. Mets au passé formel (습니다) : "날씨가 좋아요. 친구와 공원에 가요."

날씨가 좋았습니다. 친구와 공원에 갔습니다.

4. Complète : "Le matin, je bois de l'eau. Et je lis un livre." (utilise 고)

아침에 물을 마시고 책을 읽어요.

5. Traduis : "Le temps était beau. Donc j'ai pris des photos avec un ami."

날씨가 좋았어요. 그래서 친구와 같이 사진을 찍었어요.

6. Mets en formel : "바나나가 맛있어요." et "화장실이 깨끗해요."

바나나가 맛있습니다. / 화장실이 깨끗합니다.
